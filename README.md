# PDF Password Recovery Using Dictionary Attack with Hashcat

**BY Kushal Dhakal**

---

## 1. Objective
The objective of this lab is to extract a password hash from a protected PDF file and recover its password using a dictionary attack with the RockYou wordlist using Hashcat.

---

## 2. Tools Used
- Kali Linux  
- Hashcat  
- RockYou wordlist (`/usr/share/wordlists/rockyou.txt`)

---

## 3. Lab Setup
- Operating System: Kali Linux (VMware)  
- Target File: `crack_protected.pdf`  
- Wordlist: RockYou  
- Attack Type: Dictionary Attack
